Descrição

A Plus Ultra Store faz parte do meu primeiro exercicio do módulo 2 (HTML e CSS) no curso de Desenvolvimento Web Full Stack na Labenu. O objetivo foi criar uma loja online com quatro paginas:

Página Inicial | Controle de Estoque | Adicionar produtos | Funcionários

Tecnologias utilizadas

HTML 5 | CSS

Requisitos:

Inserir cabeçalho e rodapé, criar e estilizar todas as páginas utilizando apenas o CSS puro (não foi permitido o uso de bibliotecas externa).
